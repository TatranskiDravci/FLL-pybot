## Control statements and blocks
Properties:
 - K&R style blocks
 - else on the same line as if's brace
 - no parentheses around comparisons

```go
if foo == bar {
	// do stuff
} else if foo > bar {
	// do other stuff
} else {
	// do this stuff instead
}
```
